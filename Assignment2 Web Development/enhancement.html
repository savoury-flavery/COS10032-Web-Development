Enhancements used

The Enhancements Used

There have been 4 main Enhancements implemented in this website:
Menu Buttons
Embedded Youtube Video
Anchor Points
Quiz Timer


Menu Buttons

The menu buttons are located at the top of each page. There you have Home, Information, Quiz Page, Enhancements and References.
These buttons were made using both the form and button elements. By putting them together you have a functioning button that sends the user back to the linked action page.

This enhancement also uses css in the form of the .menu_button class being a hoverable. When it is hovered over it will gain a background color and the font color will change. It uses color, background color, and opacity to make it stand out with border radius giving it curved edges to give it a classic button feel.

Also implemented in this enhancement are the changes of color to links. As links have a set color for its four states, there has been css added to change these colors to ensure that the links match the websites theme.

The elements used to implement this enhacement include:
  • form
  • action
  • button
  • :link
  • :visted
  • :active
  • :hover
  • color
  • background-color
  • opacity
  • border-radius

Embedded Youtube Video

Back to Home Page

The youtube video has been embedded in the website using an a hfref link. The video is represented by a youtube logo png. This way, when you click on the image the user will be taken to the youtube video.

The elements used to implement this enhacement include:
  • a
  • href
  • img

Anchor Point

Back to Information Page

The anchor point was implemented using the id= tag on the h1 element. When the user gets to the bottom of the page they will see the link, which is another href link being used. Once that link is clicked it will take the user back to the top of the page starting at where the h1 element is located.

This enhancement is also featured at both the top and bottom of this page.

The elements used to implement this enhacement include:
  • a
  • href
  • h1

Quiz Page Timer

Back to Quiz Page

Located at the bottom on the quiz page is an enhancement that was implement to work as a 100 second quiz timer. This timer works using solely keyframe animation elements. The code goes from 0% to 100%, while counting down 100 seconds to 0 seconds. The animation is set on a 100second linear timer so that way through every percent it counts down exactly 1 second.

As the timer goes down the color of the animation shifts. It originally begins as a light purple, but as it counts down it gradually shifts to a light red. Once the counter gets to 80% or 20 seconds remaining it starts to shift into a more vibrant shade of red to signal that the timer is almost up. Once the 100 seconds pass the timer stops at 0.

This timer does not reset unless the page is refreshed, and is mainly used as a fun side game for the user while answering the quiz.

The elements used to implement this enhacement include:
  • div
  • ::after
  • content :counter(count)
  • animation
  • color
  • keyframes
